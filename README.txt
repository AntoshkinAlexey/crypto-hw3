1. Какие есть расширения для обозревателя, которые умеют взаимодействовать с Ethereum?
Metamask, Badger, Tippin.

2. Что делает MetaMask, чтобы web-приложение могло взаимодействовать с блокчейном через JavaScript?
MetaMask автоматически подписывает транзакции, обеспечивая безопасное хранение приватных ключей пользователя.

3. Каким образом обрабатываются пользовательские данные в web-приложении, чтобы они были приняты блокчейном?
При отправке данных в блокчейн, они подписываются приватным ключом, который хранится у пользователя.

4. Какие есть способы взаимодействия с блокчейном, кроме RPC-сервера и в чём их особенность?
Другой способ взаимодействия c блокчейном - использование веб-сокетов.
Необходимо установить HTTP-соединение, а затем преобразовать его в соединение веб-сокет.

5. Какие есть ещё способы получения данных из блокчейна, кроме отправка call методов и событий?
Можно использовать обозревателей блоков (explorers). Они позволяют просматривать все транзации, зарегистрированные в блокчейне.
